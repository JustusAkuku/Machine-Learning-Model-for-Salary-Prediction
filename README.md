# Machine learning Model for Salary Prediction

For this project, I downloaded the Salary Dataset and used it to build a machine learning model that makes employees' salary prediction.

# Tasks
- Data Cleaning
- Exploratory Data Analysis
- Encoding Categorical variables( Department & EducationLevel)
- Feature Engineering
- Comparison between the predicted and actual Salaries.
- Predicting salary for a new employee


  # Model Accuracy Score

- Linear Regression Algorithm
- Model Accuracy: 87.53
